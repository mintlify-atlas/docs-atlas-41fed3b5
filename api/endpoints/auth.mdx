## Initiate GitHub Login

<RequestExample>
```bash cURL
curl -X GET 'http://localhost:8000/auth/github'
```
</RequestExample>

Redirects the user to GitHub OAuth authorization page.

**Method**: `GET /auth/github`

**Authentication**: None required

### Flow

1. Generates a cryptographically secure random `state` token
2. Stores state in database with 10-minute expiration
3. Redirects to GitHub with client ID, scopes, and state

### Redirect URL

```
https://github.com/login/oauth/authorize?client_id=<CLIENT_ID>&scope=repo,read:user,user:email&state=<STATE>
```

## GitHub OAuth Callback

**Method**: `GET /auth/github/callback`

**Authentication**: None required (validates state parameter)

<ParamField query="code" type="string" required>
  Authorization code from GitHub
</ParamField>

<ParamField query="state" type="string" required>
  CSRF protection token (must match server-stored state)
</ParamField>

### Response

Redirects to `{FRONTEND_URL}/dashboard` with JWT cookie set.

### Cookie Set

```http
Set-Cookie: access_token=<JWT>; HttpOnly; Secure; SameSite=None; Max-Age=86400; Path=/
```

### Error Responses

<ResponseField name="detail" type="string">
  Error message
</ResponseField>

- **400**: Invalid or expired OAuth state
- **400**: Failed to exchange GitHub code
- **400**: Failed to fetch GitHub user

## Get Current User

<CodeGroup>
```bash cURL
curl -X GET 'http://localhost:8000/auth/me' \
  --cookie 'access_token=<your_jwt>'
```

```javascript JavaScript
fetch('http://localhost:8000/auth/me', {
  credentials: 'include'
})
  .then(res => res.json())
  .then(user => console.log(user))
```
</CodeGroup>

**Method**: `GET /auth/me`

**Authentication**: Required (JWT cookie)

### Response

<ResponseField name="id" type="integer" required>
  Internal user ID
</ResponseField>

<ResponseField name="github_id" type="integer" required>
  GitHub user ID
</ResponseField>

<ResponseField name="github_username" type="string" required>
  GitHub username
</ResponseField>

<ResponseField name="name" type="string">
  User's display name from GitHub
</ResponseField>

<ResponseField name="email" type="string">
  Primary email address
</ResponseField>

<ResponseField name="avatar_url" type="string">
  GitHub avatar URL
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp of first login
</ResponseField>

### Example Response

```json
{
  "id": 42,
  "github_id": 12345678,
  "github_username": "octocat",
  "name": "The Octocat",
  "email": "octocat@github.com",
  "avatar_url": "https://avatars.githubusercontent.com/u/12345678",
  "created_at": "2025-01-15T08:30:00.000000"
}
```

## Logout

<RequestExample>
```bash cURL
curl -X POST 'http://localhost:8000/auth/logout' \
  --cookie 'access_token=<your_jwt>'
```
</RequestExample>

**Method**: `POST /auth/logout`

**Authentication**: None required (clears cookie regardless)

### Response

```json
{
  "message": "Logged out"
}
```

The `access_token` cookie is deleted from the client.

### Notes

- Does **not** revoke the GitHub access token (user can manually revoke in GitHub settings)
- Clears the JWT cookie only; does not delete user from database
